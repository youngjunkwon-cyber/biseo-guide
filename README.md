# 나만의 Slack 비서 — 설치 가이드

회사 슬랙에서 내 앞으로 오는 멘션·DM을 AI가 요약하고 답장 초안까지 만들어주는 개인 비서 앱입니다. 본인 PC에서 구동합니다.

---

## 이 앱이 해주는 일

- 내 앞으로 오는 모든 채널 멘션 + DM 자동 수집
- 긴급 건은 즉시 DM 알림, 일반 건은 시간별 묶음 리포트 (쌓인 것 없으면 침묵)
- 답장 버튼 한 번이면 지시어 기반 AI 초안 생성 → 확인·수정 후 전송
- App Home 탭에서 미처리 / 나중에 / 답장완료 큐 전체 조회

## 시작 전 준비물

- [ ] Windows 10 이상 또는 macOS (본인 업무용 PC)
- [ ] Claude Pro 플랜 이상 계정
- [ ] Chrome 브라우저 + 회사 Slack에 로그인된 상태
- [ ] **Anthropic API 키** — 용준님(@권용준)에게 Slack DM으로 요청하면 전달해드립니다 (회사 공용 키, 개인 부담 없음)

## 설치 단계 (총 약 8분)

### ① Claude Code 설치 (3분)

- https://claude.com/claude-code 접속 → 본인 OS용 설치 파일 다운로드 → 실행
- Claude Pro 계정으로 로그인

### ② Claude for Chrome 확장 설치 (2분)

- claude.ai 접속 → Settings → Claude for Chrome → Install

### ③ 설치 프롬프트 복붙 (핵심, 3분)

Claude Code 실행 → 아래 블록 전체를 복사해서 입력창에 붙여넣고 엔터:

```
내 PC에 Slack 비서 앱을 설치해줘.

GitHub 저장소: https://github.com/youngjunkwon-cyber/slack-secretary

진행 순서:
1. 위 저장소를 내 홈 디렉토리에 클론
2. Claude for Chrome을 이용해 Slack 앱 자동 생성
   (내가 브라우저에서 "허용" 한 번 클릭)
3. Slack 토큰 자동 추출해서 .env에 기록
4. Anthropic API 키를 나한테 물어봐 (용준님한테 Slack DM으로 받은 값을 붙여넣을 거야)
5. npm install 후 npm start로 구동

중간에 필요할 때만 나한테 물어봐. 나머지는 알아서 처리해줘.
```

Claude Code가 자동 진행하며, 꼭 필요한 건 이거 두 가지:

1. **Slack 앱 생성 페이지에서 "허용" 버튼 한 번 클릭** (브라우저 자동으로 열림)
2. **Anthropic API 키 붙여넣기** — 이때 용준님한테 Slack DM으로 받은 키를 붙여넣으면 됩니다

### ④ 구동 확인 (1분)

- 터미널에 `⚡ 비서 앱 연결됨 (Socket Mode)` 로그 확인
- Slack 좌측 사이드바 "앱" 목록에 "비서" 등장 확인

## 구동 후 체크포인트

- [ ] Slack 사이드바에 "비서" 앱 표시됨
- [ ] 본인에게 DM 보내면 비서봇이 요약 카드 전송
- [ ] 카드의 `[답장]` 버튼 누르면 지시어 입력 모달 정상 오픈

## 기본 사용법

- 평소처럼 Slack 쓰기. 비서가 알아서 수집·요약
- 긴급 (시간 민감 · 직접 질문 · 의사결정 요청) → 즉시 DM 알림
- 일반 건 → 평일·주말 09~20시 매 정시 + 매일 08시 조간 브리핑
- 쌓인 것 0건이면 알림 안 옴 (침묵 규칙)
- App Home 탭 → 미처리 / 나중에 / 답장완료 전체 조회

> 📌 **꼭 기억하세요**
>
> - **PC가 켜져 있는 동안에만** 비서가 작동합니다. PC 꺼진 시간대 메시지는 재구동 시 폴링으로 최대한 보완하지만 일부 놓칠 수 있어요
> - Anthropic API 비용은 **회사 공용 계정으로 자동 결제**됩니다. 개인 부담 없음
